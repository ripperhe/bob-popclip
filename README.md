# Bob PopClip Extension

[Bob](https://bobtranslate.com) is a translation and OCR app for macOS.

[PopClip](https://popclip.app) is a macOS utility that pops up a small toolbar of action icons whenever you select text.

**This repository** holds the source code of a PopClip extension that connects the two — select any text, click the Bob icon in PopClip's popup, and Bob translates it instantly.

- Install from the [PopClip Extensions Directory](https://www.popclip.app/extensions/x/f59ba6) (requires [PopClip](https://popclip.app))
- Usage guide: <https://bobtranslate.com/guide/integration/popclip.html>

# Bob PopClip 插件

[Bob](https://bobtranslate.com) 是一款 macOS 翻译与 OCR 软件。

[PopClip](https://popclip.app) 是一款 macOS 划词工具，会在你选中文本时自动弹出一组操作图标。

**本仓库**存放的是连接两者的 PopClip 扩展的源码 —— 选中文本后点击 PopClip 菜单里的 Bob 图标，Bob 会立即翻译。

- 从 [PopClip 扩展目录](https://www.popclip.app/extensions/x/f59ba6) 安装（需要先装 [PopClip](https://popclip.app)）
- 使用说明：<https://bobtranslate.com/guide/integration/popclip.html>
